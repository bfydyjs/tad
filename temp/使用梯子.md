## 第一步，安装clash
- [点击搜索下载](https://github.com/MetaCubeX/mihomo/releases/tag/v1.19.16) mihomo-linux-amd64-v1.19.16.gz到本地并解压
- 上传到服务器
```bash
gunzip mihomo-linux-amd64-v1.19.16.gz
chmod +x mihomo-linux-amd64-v1.19.16
mv mihomo-linux-amd64-v1.19.16 ./clash
```
## 第二步，生成clash配置文件并启动clash
```bash
cd /hy-tmp
# 调用 subconverter 生成 config.yaml
curl -G "https://sub.xeton.dev/sub" \
     --data-urlencode "url=订阅链接" \
     --data "target=clash" \
     -o config.yaml
```
- 手动下载 MMDB：
[点击下载](https://github.com/Dreamacro/maxmind-geoip/releases/latest/download/Country.mmdb)
下载后放到 Clash 的工作目录，文件名为 Country.mmdb
重启 Clash。
```bash
./clash -d .
# 启动 clash，自动加载当前目录下的 config.yaml 配置文件。
```
